# 📚 Blogger App

A modern and responsive **React.js Blog Application** that fetches blog posts from a REST API and provides an interactive reading experience with search, categories, and dynamic content rendering.

## 🌐 Live Demo

**https://blogger-app-taupe.vercel.app/**

## ✨ Features

* 📱 Responsive User Interface
* 🔍 Search Blogs
* 🏷️ Category-based Filtering
* 📖 Read More / Read Less
* ❤️ Like Button
* 🖼️ Dynamic Blog Images
* ⏳ Loading State
* ⚠️ Error Handling
* 📄 Project Documentation Page

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
* REST API (Fetch API)
* React Hooks (`useState`, `useEffect`)
* React Router DOM
* Vercel (Deployment)

## 🚀 Installation

```bash
git clone <repository-url>
cd blogger-app
npm install
npm start
```

Open your browser and visit:

```
http://localhost:3000
```
